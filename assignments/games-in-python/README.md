
# 📘 Assignment: Hangman Game

## 🎯 Objective

Build a classic Hangman word-guessing game using Python input, loops, conditionals, and string handling.

## 📝 Tasks

### 🛠️ Game Setup and Word Display

#### Description

Create the initial game setup that selects a secret word and displays progress to the player.

#### Requirements
Completed program should:

- Randomly select a secret word from a predefined list
- Hide the secret word using underscores for unknown letters
- Show updated progress after each valid guess
- Allow the player to see which letters have been guessed

### 🛠️ Guess Handling and Win/Lose Logic

#### Description

Implement player input, guess validation, and game ending conditions.

#### Requirements
Completed program should:

- Accept single-letter guesses from the player
- Track correct and incorrect guesses separately
- Limit the player to a fixed number of wrong attempts
- End the game with a win message when the word is fully guessed
- End the game with a lose message and reveal the word when attempts run out
